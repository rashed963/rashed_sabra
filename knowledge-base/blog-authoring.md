# Blog Authoring

## Purpose
Create blog posts as markdown files for reliable, versioned publishing.

## Command
`npm run article:new -- --slug your-article-slug --lang ar`

## Supported languages
- `ar`
- `en`

## Workflow
1. Run the command above.
2. Open the generated file in `src/content/blog/`.
3. Update frontmatter values and article body.
4. Run the app and verify the article appears in `/blog`.
