## Temporary Portrait

`assets/portrait-temporary.jpg` is approved only for this design review. It is a `400x400` compressed image with a baked-in neural background.

Before prototype approval, replace it with:

- A recent high-resolution portrait
- At least `2400px` on the long edge
- A crop supporting desktop `4:5` and mobile `3:4`
- Confirmed publication rights

Do not use the temporary portrait as the final production asset.

## Boundary

This directory is a static mockup. It imports no production components and does not authorize prototype or production implementation.
